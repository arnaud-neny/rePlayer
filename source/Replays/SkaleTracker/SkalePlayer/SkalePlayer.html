About Skale Player library

This library contains the Skale Tracker Sound Engine. It plays SKM, XM and MOD files.

For any bug or suggestion, contact me.


Files

  /Include    -->  C++ include files
    
    SkalePlayer.h                     Main .h file, include this file into your projects
    
  
  /Lib        -->  Library files
    /Win32    
      SkalePlayer.lib                 Link this library in your project


  /Bin        -->  Binary files
    /Win32
      SkalePlayer.dll                 DLL to run your project linked with this library

    
  /Samples    -->  Sample projects
    /Win32
      SimplePlayer                    Simple player example
      RAWOutput                       Writes a 16bit stereo RAW file from a song
